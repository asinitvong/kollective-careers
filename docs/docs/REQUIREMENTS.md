# Kollective Careers - System Requirements

## Project Overview

Kollective Careers is an AI-powered Recruitment Platform that centralizes all recruitment activities from candidate application to onboarding.

---

## Objectives

- Improve candidate experience
- Reduce HR manual work
- Collect all candidate data in one database
- Support AI Resume Screening
- Support Assignment Management
- Support Interview Evaluation
- Generate Recruitment Analytics Dashboard

---

## User Roles

### Candidate
- Browse jobs
- Apply online
- Upload resume
- Complete screening questions
- Submit assignments
- Schedule interviews

### HR
- Manage job positions
- Review candidates
- Track recruitment pipeline
- Evaluate interviews
- Generate reports

### Hiring Manager
- Review candidates
- Evaluate assignments
- Submit interview feedback

### Administrator
- Manage users
- Configure system settings
- Manage AI configurations

---

## Recruitment Workflow

1. Candidate visits Career Website
2. Candidate submits application
3. AI parses resume
4. HR reviews application
5. Assignment (if required)
6. Interview
7. Offer
8. Hiring
9. Onboarding

---

## Candidate Sources

- Career Website
- Email
- LinkedIn
- JobsDB
- Facebook
- Employee Referral
